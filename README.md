# 📊 DataLens: AI-Powered Data Intelligence Studio

DataLens is a full-stack, AI-powered web application that allows users to instantly upload CSV files and automatically generate interactive charts, visualizations, and natural language insights. You shouldn't have to be a data scientist to understand your data—DataLens handles the heavy lifting of data parsing, aggregation, charting, and insight generation so you can find the signal in your noise immediately.

## ✨ Features

- **Instant CSV Parsing:** Drag-and-drop CSV uploads that are instantly parsed and analyzed client-side using PapaParse.
- **Generative Dashboard:** Automatically curates a custom dashboard of 6 beautiful, dynamic charts (Bar, Area, Pie, Donut, Scatter, and Line plots via Recharts) entirely formulated by GPT-4 and tailored to your specific dataset.
- **Natural Language "Ask Your Data":** Chat directly with your dataset. Ask questions like "Which product had the highest revenue overall?" and get plain English, rich markdown answers instantly generated by AI.
- **Raw Data Inspector:** Complete built-in searchable, paginated data table to dive directly into the raw numbers.
- **Demo Datasets:** Includes instantly loadable demo datasets (Global Energy Consumption, City Air Quality, E-Commerce Sales) so users can experience the platform immediately without needing their own CSV.
- **Premium UI/UX:** Built on Next.js 16 and Tailwind CSS with a highly dynamic, responsive, and glassmorphic aesthetic architecture utilizing rich color gradients.

## 🛠️ Technology Stack

- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS, Framer Motion (for animations)
- **UI Components:** Radix UI primitives, Lucide React (icons)
- **Data Visualization:** Recharts
- **AI Processing:** OpenAI API (`gpt-4o-mini`)
- **Data Parsing:** PapaParse

## 🚀 Getting Started

To run the DataLens platform locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Bhargavireddy16/datalens.git
   cd datalens
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up environment variables:**
   Create a `.env.local` file in the root directory and add your OpenAI API key:
   ```env
   OPENAI_API_KEY=your_openai_api_key_here
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```
   *Your application will be running at `http://localhost:3000`.*

## 📈 Demo Datasets

Three robust demo datasets are included in the `/public/demo-datasets/` folder for immediate out-of-the-box testing. Just click one of the demo dataset buttons on the homepage to instantly load up a curated set of insights!

## 🔐 Deployment

DataLens is optimized for deployment on Vercel:
1. Ensure your repository is pushed to GitHub.
2. Import the project in your Vercel dashboard.
3. Configure the `OPENAI_API_KEY` Environment Variable within your Vercel Project Settings.
4. Deploy!

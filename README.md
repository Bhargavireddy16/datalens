# DataLens: AI-Powered Data Intelligence Studio

DataLens is a full-stack, AI-powered web application that enables users to upload CSV files and automatically generate interactive visualizations and natural language insights. The platform abstracts away manual data parsing, aggregation, and charting, allowing users to immediately extract signals and insights from their raw datasets.

## Features

- **Client-Side CSV Parsing:** Drag-and-drop CSV uploads that are instantly parsed and analyzed client-side using PapaParse.
- **Generative Dashboard:** Automatically curates a custom dashboard of 6 dynamic charts (Bar, Area, Pie, Donut, Scatter, and Line plots via Recharts) formulated by GPT-4 and tailored to the dataset's specific structure.
- **Natural Language "Ask Your Data":** Chat directly with the dataset. Ask plain English questions and get context-aware, rich markdown answers generated by LLMs.
- **Raw Data Inspector:** Complete built-in searchable, paginated data table to inspect raw rows.
- **Demo Datasets:** Includes instantly loadable demo datasets (Global Energy Consumption, City Air Quality, E-Commerce Sales) for immediate evaluation.
- **Responsive Architecture:** Built on Next.js 16 and Tailwind CSS with a highly dynamic, optimized, and glassmorphic UI architecture.

## Technology Stack

- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS, Framer Motion
- **UI Components:** Radix UI primitives, Lucide React
- **Data Visualization:** Recharts
- **AI Processing:** OpenAI API (`gpt-4o-mini`)
- **Data Parsing:** PapaParse

## Local Development

To run the DataLens platform locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Bhargavireddy16/datalens.git
   cd datalens
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up environment variables:**
   Create a `.env.local` file in the root directory and add your OpenAI API key:
   ```env
   OPENAI_API_KEY=your_openai_api_key_here
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```
   *The application will be running at `http://localhost:3000`.*

## Deployment

DataLens is optimized for deployment on Vercel:
1. Ensure the repository is pushed to GitHub.
2. Import the project in the Vercel dashboard.
3. Configure the `OPENAI_API_KEY` Environment Variable within the Vercel Project Settings.
4. Deploy the application.
